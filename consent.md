# Согласие на обработку персональных данных

Настоящим я, Пользователь сервиса DocExpert AI, действуя свободно, своей волей и в своём интересе, в соответствии с Федеральным законом № 152-ФЗ «О персональных данных», даю согласие Оператору на обработку моих персональных данных на следующих условиях:

## 1. Перечень персональных данных

Данные, содержащиеся в загружаемом мной документе, которые могут включать:
- Фамилия, имя, отчество
- Паспортные данные
- Адрес регистрации и проживания
- ИНН, СНИЛС
- Банковские реквизиты
- Иные данные, содержащиеся в документе
- Идентификатор пользователя Telegram (ID чата)

## 2. Цели обработки

- Автоматизированный анализ загруженного документа с помощью ИИ
- Предоставление отчёта с результатами анализа

## 3. Действия с персональными данными

Сбор, запись, извлечение, передача (YandexGPT API), удаление.

## 4. Передача третьим лицам

Данные передаются в Yandex Cloud (ООО «Яндекс.Облако») исключительно для целей анализа текста с помощью YandexGPT. Обработка Yandex Cloud осуществляется на серверах на территории РФ.

## 5. Срок действия согласия

Согласие действует с момента его предоставления до момента отзыва.

## 6. Отзыв согласия
