# Политика конфиденциальности DocExpert AI

*Дата вступления в силу: 06.06.2026*

## 1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») разработана в соответствии с Федеральным законом № 152-ФЗ «О персональных данных» и определяет порядок обработки и защиты информации о пользователях сервиса DocExpert AI (далее — «Сервис»).

Используя Сервис, вы подтверждаете ознакомление с настоящей Политикой и выражаете согласие с её условиями.

## 2. Какие данные мы обрабатываем

Сервис обрабатывает исключительно данные, добровольно предоставленные пользователем в составе загружаемого документа:

- Фамилия, имя, отчество
- Паспортные данные
- Адрес регистрации
- ИНН, СНИЛС
- Банковские реквизиты
- Иные персональные данные, содержащиеся в загруженном документе
- Telegram ID пользователя (технический идентификатор чата)

## 3. Цели обработки

Данные обрабатываются исключительно для:

- Анализа загруженного документа с помощью ИИ (YandexGPT)
- Предоставления пользователю отчёта об анализе

Данные **не используются** для рекламы, маркетинга, передачи третьим лицам (кроме указанных в п. 5) или создания профилей пользователей.

## 4. Принципы обработки

- Обработка осуществляется только в объёме, необходимом для анализа документа
- Данные обрабатываются на серверах, расположенных на территории Российской Федерации
- Сервис **не хранит** загруженные файлы и извлечённый текст после завершения анализа
- Временные файлы автоматически удаляются сразу после обработки
- Логи сервера содержат только техническую информацию (название файла, количество символов) и не включают персональные данные

## 5. Передача данных третьим лицам

Для анализа текста используется API YandexGPT (ООО «Яндекс.Облако», Яндекс 360). Текст документа передаётся в Yandex Cloud на время обработки.

Правовые основания: Yandex Cloud действует как обработчик данных в соответствии с поручением оператора (ст. 6, 152-ФЗ). Данные обрабатываются на серверах Yandex Cloud на территории РФ.

Пользователь уведомлён и соглашается с передачей текста документа в YandexGPT для целей анализа.

## 6. Безопасность

Сервис применяет следующие меры защиты:

- Все соединения используют шифрование (TLS/HTTPS)
- Файлы хранятся во временной директории и удаляются немедленно после обработки
- Доступ к серверу ограничен
- Yandex Cloud сертифицирован по стандартам безопасности РФ

## 7. Сроки обработки

- Загруженный файл: удаляется сразу после извлечения текста (в течение нескольких секунд)
- Извлечённый текст: удаляется после получения результата анализа
- Результат анализа: отправляется пользователю и не сохраняется на сервере

## 8. Права пользователя

В соответствии с 152-ФЗ пользователь имеет право:

- Получить информацию об обрабатываемых данных
- Отозвать согласие на обработку
- Требовать удаления данных
